# PJ003 Meta-Log 작업 목록 (PM 에이전트)

## 프로젝트 목표
MAS(멀티 에이전트 시스템) 자체의 성장 과정을 기록하는 "Meta-Log" 시스템을 구축합니다. 이는 사용자의 AI 에이전트 활용 여정을 보여주는 대시보드이자 포트폴리오 역할을 합니다.

## 워크플로우 규칙 (MAS)
1.  **PM 에이전트**: 가설 수립 -> 사용자 스토리 작성 -> 스토리보드/UX 경험 설계 -> 명세서 작성.
2.  **디자이너 에이전트**: 디자인 시스템 정의 -> 와이어프레임 제작 -> UI 컴포넌트 설계.
3.  **개발자 에이전트**: 구현 -> 테스트 -> 배포.
